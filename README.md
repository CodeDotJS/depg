<p align="center"><b>Command line tool to find total dependents on a repository!
</b></p>


## Install

```

$ npm install --global depg
```
__`OR`__
```
$ sudo npm install --global depg
```

## Preview

<p align="center">
<img src="media/depg.gif" alt="depg">
</p>


## Usage


```
Usage: depg <username/org> <repository>

Example:

 $ depg facebook react

```
